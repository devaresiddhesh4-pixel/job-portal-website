# Job Portal Website

A responsive Job Portal Website built using HTML, CSS, and JavaScript that enables users to browse and search job opportunities through an interactive and user-friendly interface.

## Features

* View job listings
* Search jobs by title
* Search jobs by company name
* Responsive design
* Interactive user interface
* Modern job card layout

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Description

This project is a frontend-based Job Portal Website designed to help users explore available job opportunities. Users can search for jobs using keywords and view job details such as company name, location, and salary package.

The project demonstrates key frontend development concepts including DOM manipulation, event handling, responsive design, and JavaScript array methods.

## Learning Outcomes

* DOM Manipulation
* Event Handling
* Search Functionality
* Responsive Web Design
* JavaScript Array Methods
* Frontend Development
